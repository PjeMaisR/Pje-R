PJe+R

Visão Geral

Extensão de navegador com melhorias ao "microsistema PJe". A extensão `PJe+R` ou `pjemaisr` busca aumentar a racionalidade, a rapidez e a relevância das opções apresentadas no sistema PJe, de forma a assegurar mais acessibilidade, eficiência e melhoria da experiência do usuário. É um projeto colaborativo, com a atuação de Caio, Israel Azevedo Fabiano, Ageílson Silva e Igor, inspirado pelo uso de scripts de usuário (como os utilizados pelo `GreaseMonkey` e assemelhados) e pela extensão `SEI++`. Mais recentemente, o projeto passou a inspirar-se também na experiência da extensão `AVJT - Assistente Virtual da Justiça do Trabalho`, de Sisenando Silva.

Instalação

A versão para o Firefox está hospedada na loja Firefox Browser Add-ons.

A versão para o Chrome está na loja Chrome Web Store.
